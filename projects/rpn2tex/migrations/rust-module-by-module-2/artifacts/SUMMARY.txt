================================================================================
KEY FINDINGS
================================================================================

SUPPORTED OPERATORS:
- Addition (+) -> rendered as "+"
- Subtraction (-) -> rendered as "-"
- Multiplication (*) -> rendered as "\times"
- Division (/) -> rendered as "\div"

NOT SUPPORTED:
- Exponentiation (^) -> LexerError

PRECEDENCE RULES DISCOVERED:
- Level 2 (Higher): * and / 
- Level 1 (Lower): + and -
- Left-associative for operators of same precedence

PARENTHESIZATION RULES:
- Left operand: Add parens if operand's precedence < current precedence
- Right operand: Add parens if operand's precedence <= current precedence

FLOATING-POINT SUPPORT:
- Decimal numbers fully supported (e.g., 3.14, 1.5)
- Decimal representation preserved in output

OUTPUT FORMAT:
- All LaTeX expressions wrapped in: $...$
- Single space around operators
- Space inside parentheses: "( expr )"
