=============================================================================
KEY IMPLEMENTATION DETAILS
=============================================================================

TOKEN RECOGNITION:
  Lexer recognizes '+' and emits Token(PLUS, "+", line, column)

RPN PARSING:
  Stack-based evaluation:
  1. Push numbers onto stack
  2. When '+' encountered, pop right, pop left
  3. Create BinaryOp("+", left, right)
  4. Push result back

LATEX GENERATION:
  Recursive traversal: "left + right" with spaces
  
AST STRUCTURE:
  "1 2 + 3 +" produces:
  BinaryOp("+", BinaryOp("+", Number("1"), Number("2")), Number("3"))
