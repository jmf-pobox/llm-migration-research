PRECEDENCE RULES
================
1. Multiplication (*) and Division (/) have HIGHER precedence
2. Addition (+) and Subtraction (-) have LOWER precedence
3. All operators are LEFT-ASSOCIATIVE
4. Parentheses are added when lower-precedence operations appear as operands of higher-precedence operations

OUTPUT FORMAT
=============
All valid outputs are wrapped in LaTeX math mode: $ ... $
Spaces surround all operators in the output.
Parentheses appear as ( and ) with spaces before and after content.

- RPN (Reverse Polish Notation) input is converted to infix notation with proper parenthesization
- All numbers (integers and floats) are preserved as-is in output
